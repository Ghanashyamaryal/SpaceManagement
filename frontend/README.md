# React Starter Kit

A modern, scalable React starter kit built with **TypeScript**, **Zustand**, **Tailwind CSS v4**, and a collection of reusable **custom hooks** and **utility functions**.

Designed to reduce boilerplate and provide a clean foundation for real-world applications.

---

## Features

*  **React + TypeScript** setup
*  **Tailwind CSS v4** with design tokens (CSS variables)
*  **Zustand** for lightweight state management
*  **Persisted state** (localStorage & sessionStorage)
*  **Immer integration** for simpler state updates
*  **Reusable custom hooks**
*  **Utility functions** (date & number formatting)
*  **Dark / Light / System theming**
*  Clean and scalable folder structure

---

## Tech Stack

| Category      | Technology      |
| ------------- | --------------- |
| Framework     | React           |
| Language      | TypeScript      |
| Styling       | Tailwind CSS v4 |
| State Mgmt    | Zustand         |
| Middleware    | Persist, Immer  |
| UI Components | shadcn/ui       |

---

## Project Structure

```bash
src/
├── components/        # UI components (buttons, inputs, etc.)
├── hooks/             # Custom hooks
├── lib/utils/         # Utility functions
├── providers/         # Context providers (theme)
├── store/             # Zustand stores
```

## Getting Started

